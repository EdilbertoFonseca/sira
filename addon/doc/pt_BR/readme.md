## Descrição

O **SIRA** (_Sistema Interno de Ramais e Anotações_) é um complemento para o NVDA desenvolvido para otimizar a comunicação interna e o fluxo de informações na Secretaria de Saúde.

Com foco em acessibilidade e agilidade, o SIRA oferece:

* **Consulta de Ramais:** Cadastro e busca rápida de contatos de todos os setores;
* **Logística de Transporte:** Registro de recados específicos para o setor de transportes;
* **Gestão Hospitalar:** Registro e acompanhamento de altas médicas;
* **Central de Recados:** Gerenciamento centralizado de mensagens internas.

A interface foi projetada para total compatibilidade com o **NVDA**, garantindo uma experiência inclusiva, fluida e eficiente para usuários com deficiência visual em ambientes administrativos.

## Instalação

1. No NVDA, acesse o menu **Ferramentas** e selecione **Loja de Complementos**.
2. Na aba **Complementos Disponíveis**, utilize o campo **Pesquisar**.
3. Digite "SIRA" e pressione **Enter**.
4. Selecione o complemento na lista e clique em **Instalar**.
5. Reinicie o NVDA para aplicar as alterações.

## Comandos e Atalhos

| Ação                                                  | Atalho                   |
| :---------------------------------------------------- | :----------------------- |
| Exibir lista de ramais e contatos cadastrados         | Alt+1 (Teclado Numérico) |
| Registrar mensagens para o setor de transporte        | Alt+2 (Teclado Numérico) |
| Registrar alta hospitalar                             | Alt+3 (Teclado Numérico) |
| Registrar mensagens gerais para qualquer departamento | Alt+4 (Teclado Numérico) |

> **Nota:** Os atalhos podem ser personalizados em: _Menu NVDA > Preferências > Gestos de Entrada_.
