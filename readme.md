## Description

**SIRA** (_Sistema Interno de Ramais e Anotações_) is an NVDA add-on developed to enhance internal communication and information logging within the Health Department.

Focused on accessibility and efficiency, SIRA allows you to:

* Register and consult telephone extensions for all sectors;
* Log messages intended for the transport sector;
* Note medical discharges for monitoring and follow-up;
* Manage general memos and internal messages in a centralized way.

The interface is fully compatible with screen readers like **NVDA**, ensuring a smooth and inclusive experience for visually impaired users.

Ideal for daily use in public and administrative environments, **SIRA** contributes to more agile, precise, and organized information management.

## Installation

1. In NVDA, open the **Tools** menu and select **Add-on Store**.
2. In the **Available Add-ons** tab, go to the **Search** field.
3. Type "SIRA" and press **Enter**.
4. Select the add-on and click **Install**.
5. After installation, restart NVDA to complete the process.

## Commands and Shortcuts

| Action | Shortcut |
| :--- | :--- |
| Displays a window with all contacts registered in the Extension Registry System | Alt+1 (Numpad) |
| Log messages for the transport department | Alt+2 (Numpad) |
| Log hospital discharges | Alt+3 (Numpad) |
| Allows logging messages for all departments | Alt+4 (Numpad) |

> Shortcuts can be customized in the NVDA **Input Gestures** menu.
